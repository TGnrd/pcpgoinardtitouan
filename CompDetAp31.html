Chargement des utilisateurs

Voici le code des fonctions permettant de charger les utilisateurs GitLab ainsi que le rendu visuel.

  • Affichage des utilisateurs — getUsers

    Fait appel à EraseHTMLTab et CreateTableRow.

  • Suppression du tableau — EraseHTMLTab

    Parcourt le tableau des utilisateurs pour supprimer chaque ligne.

  • Création des lignes — CreateTableRow

    Pour un utilisateur donné, génère le code HTML de la ligne correspondante.

Activation / Désactivation des utilisateurs

Ces fonctions font appel à l'API REST de GitLab pour changer l'état d'un utilisateur.

  • activerDesactiverUser — détermine si l'utilisateur est activé ou désactivé.
  • changerEtatUser — change l'état de l'utilisateur selon son état actuel.
  • À la fin, getUsers est appelée pour rafraîchir l'affichage.

Limite du nombre de projets

Ces fonctions font appel à l'API REST de GitLab pour modifier la limite de projets d'un utilisateur.

  • changerLimite — récupère la valeur saisie dans le champ texte.
  • changerNombreLimite — met à jour la limite pour un utilisateur donné.
  • À la fin, getUsers est appelée pour rafraîchir l'affichage.
