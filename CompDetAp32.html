Missions

Réalisation du diagramme de classe — il s'agit de la plus grosse partie du premier milestone.

Création du contrôleur d'organisations avec Symfony.

  • bin/console make:entity "Organisation" — génère l'entité depuis la console Symfony.
  • OneToMany — une Organisation possède plusieurs Contacts.
  • ManyToOne — plusieurs Contacts appartiennent à une seule Organisation.
  • Règles de validation — code postal, numéro de téléphone et URL via Symfony Assert.

Code de l'entité et règles de gestion

Entité Organisation générée par Symfony

Expressions régulières utilisées

Code postal :
^(?:[0-9]{2}|2A|2B)[0-9]{3}$

Téléphone :
^(?:(?:\+|00)33|0)\s*[1-9](?:[\s.-]*\d{2}){4}$

URL / Email :
#[Assert\Email()]
